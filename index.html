Intro

Pure plain is a mathimatical theori about how dimentions, numbers and manny fysical properties work. It is in a understading about that there may is an additional dimestion within every one. These dimentions are made out of a "mode" called 'Plain'

To understand simply think about a piece of papper, on one side there is a 2d grid. But there is also a backside to the papper. Pure plain then states that there is a simular 2d grid with some changes. If you then would pull that papper apart so that no sides where touching but still connected, you would have a 'Pure plain' of that papper.
